Readme

To compile the project the following has to be done:

  • Unzip the project file
  • Open a commandline tool and move to the unzipped folder
  • Run the "ant" command from the commandline, this should compile and run the program (tested on linux and windows)
  • The "ant clean" command will clean and remove the compiled files
  • If the project is already compiled you can use the "ant run" command to run
  • If the program is not working you might have to update your graphics card to the newest drivers, and/or download the
    JOGL natives for your system and put them in the lib folder (windows, linux and mac os natives should already be there,
    but we have only tested for linux and windows).

The mandelbrot set should show up on the screen.
There are two files showing our recording functionality under the documents folder. One GIF and one MP4 file.
